Setup Wizard

Eight steps. Almost entirely automated.

Watch the progress bars. You're only needed for your API keys and GitHub login.

1

Welcome

Detects hostname and architecture. Personalizes the greeting.

2

System Check

Verifies Docker, Git, RAM, disk, internet. Auto-fixes failures with a live log stream.

3

Cloudflare Tunnel

Shows your remote URL the moment it's ready — with a QR code. Upgrade to a stable named tunnel optionally.

4

GitHub

OAuth connect. Enables repo browser, PR creation, and the inline git panel.

5

AI Providers

Auto-detects keys from the environment. Test and save Anthropic, OpenAI, Google, or Ollama keys.

6

opencode

Installs the opencode AI coding agent. Streams install progress live.

7

NanoClaw

Clones and builds the Claude agent Docker container. Streams output live.

8

Complete 🎉

Confetti, both URLs with QR codes, and a button to open the dashboard.

Features

Everything in the browser.

No SSH. No command line. The app does the work.

⌨️

AI Coding IDE

Full terminal powered by opencode — a PTY session in your browser via WebSocket. Works with Claude, GPT-4, Gemini, or local Ollama models. Per-project sessions.

💬

AI Agent Chat

Streamed Claude responses via NanoClaw running in Docker. Markdown rendering, syntax-highlighted code blocks, project-scoped context.

📊

Real-Time Dashboard

CPU, RAM, disk, Pi temperature, and uptime via SSE — updated every 2 seconds. Service status cards, GitHub activity feed, and quick-launch projects.

🐙

GitHub Integration

Browse repos, import with one click (streams clone progress), create PRs, view activity — all without leaving the app. Full git panel in the IDE sidebar.

☁️

Cloudflare Tunnel

Remote access from anywhere — no port forwarding, no dynamic DNS. Quick tunnel works immediately. Upgrade to a named tunnel for a stable URL.

🔐

Encrypted Key Storage

API keys are AES-256-GCM encrypted at rest using a machine-derived key (hostname + primary MAC). Never stored in plaintext. Standard Go crypto package only.

Dual Access

LAN and internet, simultaneously.

Both URLs are live from the moment the installer finishes. No configuration needed.

Local Network

http://raspberrypi.local:3000

Zero-latency access on your home or office LAN. Uses mDNS — works on any device on the same network.

Anywhere (Cloudflare)

https://random-name.trycloudflare.com

Full internet access over TLS. Set up automatically by the installer. Upgrade to a stable named URL in the wizard.

Architecture

One binary. No runtime.

The production binary is a single statically-linked Go binary with the Vue 3 client embedded via //go:embed. No Node.js runtime in production.

Browser (Vue 3) ←── HTTP / WebSocket / SSE ──→ Go server (Chi, :3000) │ ┌─────────────────┬──────────────┬──────────────┐ opencode NanoClaw cloudflared SQLite (creack/pty) (Docker + (child proc, (projects, per project) SQLite bridge) quick tunnel) settings, messages)
